# SUPER HELLO

SUPER HELLOは、Hello! Project / ハロプロのイベント候補、参加予定、チケット支払い、グッズ、遠征、参加履歴、MCY連携出力を管理する静的アプリです。

v1.0 基本完成版です。HTML / CSS / JavaScript のみで動作し、サーバー、DB、ログイン、外部APIは使用しません。

## v1.0 時点の機能

- イベント候補の登録、編集、削除、検索、状態フィルター、JSON出力
- イベント情報の取込補助とイベント候補フォームへの反映
- イベント候補から参加予定への昇格
- 参加予定・申込情報の編集、削除、検索、状態フィルター、JSON出力
- 参加予定からチケット支払い情報の作成
- チケット代、手数料、支払い状態、MCY連携状態の管理
- 申込締切、当落発表日、支払期限、公演日の期限・リマインド管理
- グッズ・物販支出の登録、編集、削除、JSON出力
- 遠征費の登録、編集、削除、JSON出力
- 参加履歴・感想ログの登録、編集、削除、JSON出力
- MCY用CSV / JSON出力
- MCY連携対象のプレビューと表示分を連携済にする機能
- 全データJSONバックアップ出力
- 全データJSON復元
- 開発・確認用サンプルデータ投入
- SUPER HELLO全データ削除（二重確認）

## 使い方

1. `index.html` をブラウザで開きます。
2. 「イベント情報 取込補助」または「イベント候補 登録・編集」からイベント候補を登録します。
3. イベント候補一覧の「参加予定へ」から参加予定を作成します。
4. 参加予定一覧の「支払いへ」からチケット支払いを作成します。
5. グッズ、遠征、参加履歴は各登録・編集フォームから登録します。
6. MCY連携 出力で対象を確認し、CSVまたはJSONを出力します。
7. 出力後、必要に応じて「表示分を連携済にする」を実行します。
8. 定期的に「全データJSONバックアップ出力」でバックアップを保存します。

## データ保存方式

データはブラウザの `localStorage` に保存します。

主な保存キー:

- `superHelloEventsV01`
- `superHelloParticipationsV02`
- `superHelloPaymentsV03`
- `superHelloMcyExportLogsV04`
- `superHelloGoodsV07`
- `superHelloTravelsV08`
- `superHelloReviewsV09`

v1.0では既存キーを変更していません。ブラウザのサイトデータ削除、別ブラウザ利用、プライベートブラウズではデータが失われる場合があります。

## バックアップ / 復元

「バックアップ・復元 / 保守」セクションから全データを一括バックアップできます。

- ファイル名: `super-hello-backup-YYYYMMDD.json`
- 対象: SUPER HELLO関連の `localStorage` データ
- 復元時はSUPER HELLO用バックアップJSONか最低限チェックします。
- 復元すると現在のSUPER HELLO関連データを上書きします。
- 復元前には現在データのバックアップ出力を推奨します。

## MCY連携

SUPER HELLOで発生したチケット代、グッズ代、遠征費をMCYの日々収支へ連携するため、MCY用CSV / JSONを出力できます。

MCY連携対象:

- チケット支払い
- グッズ支出
- 遠征支出

MCY連携対象外:

- イベント候補
- 参加予定
- 参加履歴・感想ログ

MCY連携状態は `未連携`、`連携済`、`対象外` を管理します。プレビューに表示された対象は「表示分を連携済にする」で一括更新できます。
