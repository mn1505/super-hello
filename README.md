# SUPER HELLO

Hello! Project / ハロプロのイベント候補、参加予定、支払い、グッズ・物販、遠征、MCY連携出力を管理する静的アプリです。

## v0.9 動作確認項目

- 参加履歴を新規登録できること。
- イベントに紐付けた履歴がイベント名・開催日・会場付きで表示されること。
- 参加予定から「履歴へ」ボタンで履歴作成できること。
- 同じ参加IDで二重作成しようとすると既存履歴編集になること。
- 履歴作成時に参加状態を参加済へ更新できること。
- 座席、整理番号、満足度、印象曲、感想を保存できること。
- 履歴を編集できること。
- 履歴を削除できること。
- 履歴一覧で検索・フィルターできること。
- グループフィルター、満足度フィルター、タグ検索ができること。
- 履歴JSON出力ができること。
- 表示件数、平均満足度、最高満足度、最新参加日が表示されること。
- 参加履歴・感想ログがMCY連携プレビュー、MCY用CSV/JSONに含まれないこと。
- 既存のイベント候補、参加予定、支払い、期限管理、グッズ管理、遠征管理、MCY連携が壊れていないこと。
- localStorage キー `superHelloReviewsV09` に参加履歴が保存され、既存キー `superHelloEventsV01`、`superHelloParticipationsV02`、`superHelloPaymentsV03`、`superHelloMcyExportLogsV04`、`superHelloGoodsV07`、`superHelloTravelsV08` が変わっていないこと。

## v0.8 動作確認項目

- 遠征を新規登録できること。
- 交通費・宿泊費・その他費用から合計が表示されること。
- 遠征を編集できること。
- 遠征を削除できること。
- 遠征一覧で検索・予約状態フィルター・MCY連携状態フィルターが使えること。

- イベント未紐付けでも登録できること。
- 遠征JSON出力ができること。
- 遠征費がMCY連携プレビューに表示されること。
- MCY用CSV/JSONに遠征費が含まれること。
- 表示分を連携済にすると、遠征のMCY連携状態も連携済になること。
- 既存のイベント候補、参加予定、支払い、期限管理、グッズ管理、MCY連携が壊れていないこと。
- localStorage キー `superHelloTravelsV08` に遠征情報が保存され、既存キー `superHelloEventsV01`、`superHelloParticipationsV02`、`superHelloPaymentsV03`、`superHelloMcyExportLogsV04`、`superHelloGoodsV07` が変わっていないこと。

## v0.5 動作確認項目

- 「イベント情報 取り込み補助」に公式サイト等からコピーした文章を貼り付け、「解析」でプレビューが表示されること。
- プレビューでイベント名、開催日、開演時間、会場、都道府県、グループ、種別、チケット価格、申込締切、当落発表日、支払期限、情報源URL、メモを確認できること。
- 「イベント登録フォームへ反映」を押すまで、イベント候補は登録されず localStorage も更新されないこと。
- 反映後、既存のイベント候補登録フォームに抽出できた項目だけが入ること。
- チケット価格は既存イベント候補に専用項目がないため、メモへ「チケット価格: xxxx円」として反映されること。
- 反映後、既存の「登録」ボタンでイベント候補を保存でき、一覧・検索・JSON出力が従来通り動くこと。
- 参加予定、支払い、MCY連携出力の既存操作が従来通り動くこと。
- localStorage キー `superHelloEventsV01`、`superHelloParticipationsV02`、`superHelloPaymentsV03`、`superHelloMcyExportLogsV04` が変わっていないこと。
